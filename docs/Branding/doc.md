## Source Signals

- Main site positions Vinpix as an independent software studio for products, games, and AI-powered workflows.
- Business page positions Vinpix Studio as a B2B delivery partner for German SMEs and consulting teams, focused on manual workflow automation and internal tools in 2-4 weeks.
- Product/game pages include Kitchen Together and other game portfolio entries, with a playful but direct tone.
- Visual language in code uses high-contrast black/white, off-white `#F0F0F0`, bold uppercase typography, thick borders, mono utility labels, and dither/interactive motion accents.
